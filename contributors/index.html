How to cite

Citing the software

Please use the following citation to reference the redbKIT software and its online documentation

F. Negri. redbKIT Version 2.2. http://redbkit.github.io/redbKIT/, 2016.

(you should fill in the version you used, rather than the example version above)

BibTex entry

@Misc{redbKIT,
author  = {Negri, F.},
title   = {redb{KIT} {V}ersion 2.2},
howpublished = {\url{http://redbkit.github.io/redbKIT/}},
year = {2016}
}
